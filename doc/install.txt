you need these packages:

Required packages:
sudo apt install
libmariadbclient-dev
libmariadb-dev-compat
libjansson-dev 
p7zip p7zip-full
libmodbus-dev
libcurl4-openssl-dev
gnuplot
nmap
libusb-dev
cmake

Optional packages:
matchbox (on screen keyboard)

Clone the repo:
git clone ssh://jeindhoven@10.0.0.33/home/jeindhoven/git/tcFarmControl

And manually install: libsnap7 (1.4.2)
wget https://sourceforge.net/projects/snap7/files/1.4.2/
7z x snap7-full-1.4.2.7z
cd snap7-full-1.4.2/build/unix
make -f arm_v7_linux.mk all
sudo make -f arm_v7_linux.mk install

Or 64 bit raspbian:
https://sourceforge.net/projects/snap7/files/Snap7-IoT/snap7-iot-arm/
tar xvzf snap7-iot-arm-1.4.2.tar.gz
cd snap7-iot-arm-1.4.2/build/unix
make -f arm_v7_x64_linux.mk
sudo make -f arm_v7_x64_linux.mk install

Building farmcontrol:
cd build
cmake ../src
make

Postbuild config:
Add to /etc/fstab:
tmpfs   /tmp    tmpfs   nodev,nosuid,size=1G    0   0
restart

Add service:
sudo cp tcFC.service /etc/systemd/system
maybe change the user and dir in the system file
sudo systemctl enable tcFC.service
sudo systemctl start tcFC.service
